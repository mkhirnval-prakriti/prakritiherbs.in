Create a complete Lead Tracking + Full Data Export System for Call and WhatsApp tracking with detailed Excel reporting.

========================
OBJECTIVE
=========

Track every user interaction (Call clicks, WhatsApp clicks, missed calls, answered calls) and store FULL RAW DATA.
Admin panel will show limited fields, but Excel export must contain COMPLETE detailed data.

========================
FRONTEND TRACKING
=================

1. On "Call Now" click:

   * Generate eventId
   * Capture:

     * source
     * page_url
     * user_agent
     * timestamp
   * Fire Meta Pixel:
     fbq('track', 'Contact', { eventID: eventId, channel: 'call' })
   * Send API request to /api/lead-click

2. On "WhatsApp Now" click:

   * Generate eventId
   * Capture:

     * source
     * page_url
     * user_agent
     * timestamp
   * Modify link:
     https://wa.me/91XXXXXXXXXX?text=Hi Source:{source}_{eventId}
   * Fire Pixel
   * Send API request

========================
CALL TRACKING (BACKEND)
=======================

Integrate call tracking API (Exotel/Twilio):

Capture:

* customer_phone
* call_status (missed / answered)
* call_duration
* call_time

========================
DATABASE STRUCTURE
==================

Table: lead_tracking

Fields (FULL DATA STORAGE):

* id

* source (TAJ / FB / Direct)
* customer_phone

* call_duration
* page_url
* landing_page
* campaign_name (if available)
* adset_name (if available)
* ad_name (if available)
* device_type
* browser
* ip_address
* user_agent
* referrer
* city
* state
* country
* created_at
